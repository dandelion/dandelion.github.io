Dandelion is a free and Open Source Java framework which aims to ease web development in Java-based environments.

It helps to manipulate the web assets (JS, CSS) in many ways: good organization and classical optimizations like compression and aggregation. It also facilitates the integration of JavaScript libraries thanks to a set of extensible components.

  Assets organization

Cherish your web resources! Dandelion offers a new way of dealing with web assets. Organize them into scopes and build a real asset stack.

  Assets optimization

Apply the best pratices of web development without effort! Web assets can be minified, aggregated and optimally injected into web pages, thus improving page load times!

  Multiple locations

Collect assets from multiple locations before using them: webapp, classpath, webapp, CDNs or WebJars. Dandelion also support dynamically generated assets!

  Template engine integration

Dandelion and its components all provide a JSP taglib. And if you're cool people who use cool technologies, we also provide support for Thymeleaf!

  Ready-to-use components

The most advanced component is Dandelion-Datatables which allows you to create awesome HTML tables using DataTables. Other components are also incubating...

  Easily extensible

Dandelion makes use of the Service Provider Interface API introduced in Java 6 to provide plenty of extension points.
