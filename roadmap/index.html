Roadmap

Note that this is not a formal roadmap, more a priority list which evolves with time.

Dandelion

  • v0.3.0

    • Assets compression
    • Assets aggregation
    • Template scopes
    • New @Scope annotation allowing to activate a scope request-wide

  • v0.4.0

    • Conditional assets (IE7+, IE8+, ...)
    • Assets exotiquess

  • Later

    • Web configurator
    • Extractor Maven plugin
    • Yeoman integration

Dandelion Datatables

  • v0.9.1

    • Support for Bootstrap 3 and other Bootstrap themes (see Tablecloth)
    • 0.9.0 fixes

  • v0.10.0

    • Websocket support for continuously-updating table
    • Assets exotiquess

  • v0.11.0

    • We've planned to make the connection between Dandelion-Datatables and Dandelion (core module) in this release
    • Add support for Spring Data REST (server-side processing)
    • Assets exotiquess

  • v0.12.0

    • Add support for live-editing table
