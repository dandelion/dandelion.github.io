Roadmap

Note that this is not a formal roadmap, more a priority list which evolves with time.

Dandelion

  • v0.3.0

    • Assets compression, full or partial compression management
    • Assets aggregationn full or partial aggregation management
    • Template scopes (examples of assets scopes)
    • New @Scope annotation allowing to activate a scope request-wide

  • v0.4.0

    • Conditional assets (IE7+, IE8+, ...)
    • Preprocessors for assets (Less, CoffeeScript, ...)

  • Later

    • Web configurator, a kind of web app allowing to visualize/build assets trees
    • Asset Extractor Maven plugin, allowing to extract available assets from the classpath (webapp, resources, webjars)
    • Bower integration

Dandelion Datatables

  • v0.9.1

    • Support for Bootstrap 3 and other Bootstrap themes (see Tablecloth)
    • 0.9.0 fixes

  • v0.10.0

    • Websocket support for continuously-updating table

  • v0.11.0

    • We've planned to make the connection between Dandelion-Datatables and Dandelion (core module) in this release
    • Add support for Spring Data REST (server-side processing)

  • v0.12.0

    • Add support for live-editing table

Dandelion Webanalytics

  • v0.3.0

    • Improve support for Google Analytics (web tracking API)

New components

We've planned to develop new components, such as:

  • Dandelion Glyph
  • Dandelion Forms
  • If you have any suggestion, feel free to contact us!
