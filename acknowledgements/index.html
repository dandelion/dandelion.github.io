Dandelion (and all components) are tested using the great Java profiling tool YourKit.

YourKit is kindly supporting open source projects with its full-featured Java Profiler. YourKit, LLC is the creator of innovative and intelligent tools for profiling Java and .NET applications.

Take a look at YourKit's leading software products: YourKit Java Profiler and YourKit .NET Profiler

We use JRebel to boost our development. It's a terrific tool created by Zeroturnaround which allow you to view code changes instantly. Many thanks to them for giving us an OSS licence.

Cloudbees in one of the best existing PaaS solutions. We mainly use Jenkins, the famous open source Continuous Integration server included in Cloudbees in the way that everytime a commit is pushed in a GitHub repository, a build is launched in Jenkins.

Trello helps you to organize everything. It's a great tool for software project management. There are lots of features and a couple of native mobile apps. In short, since we started using it, we never stopped. :-)

Thanks to JetBrains for giving us an OSS licence!

generatedata.com is a free, open source tool written in JavaScript, PHP and MySQL that lets you quickly generate large volumes of custom data in a variety of formats for use in testing software, populating databases, and... so on and so forth. Many thanks to Benjamin Keen for developing this great tool!
