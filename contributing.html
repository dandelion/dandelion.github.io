Contributing to the documentation

One would think you want to improve the page. We suggest that you use prose.io for this. It's very simple!

By clicking the link below, you will be redirected to prose.io to edit this page. Just follow these steps:

  • Sign in with your Github account
  • Make some changes
  • Submit the changes

Once done, it will generate a pull request in the right repository. And that's all!
