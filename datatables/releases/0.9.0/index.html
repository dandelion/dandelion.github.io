Dandelion-Datatables 0.9.0 change log??-??-2013

Core

  • AJAX sources

    From this release, you have to configure yourself the URL used to populate your AJAX-enabled tables (the context path won't be prepended any longer). All existing tutorials have been updated accordingly.

  • More tests

    More unit tests and integration tests have been added.

JSP implementation

  • New {% refjsp dt table titleKey %} table attribute, allowing to use a resource key used to lookup the title value of a column.
  • Removed the labels table attribute (nouvelle gestion de l'i18n)

Thymeleaf implementation

  • New {% reftl dt table dt:dom %} table attribute
  • New {% reftl dt th dt:selector %} th attribute

Closed issues

  • {% issue 100 %}: Fix base.url configuration point
  • {% issue 101 %}: Add support for cssStyle and cssClass when using AJAX source
  • {% issue 102 %}: "TypeError: oColumn is undefined" on DOM when data is a empty list
  • {% issue 103 %}: Support dynamic value for the column's "visible" attribute
  • {% issue 105 %}: Not all existing properties did appear as "valid"
  • {% issue 106 %}: Add support for sDom parameter
  • {% issue 107 %}: Add support for sSelector parameter of the ColumnFilter plugin
  • {% issue 109 %}: Conflict between the Bootstrap2 theme and basic features enablement
