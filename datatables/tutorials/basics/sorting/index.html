Sorting data

You can alter the sorting characteristics in many ways:

  • You can enable/disable global sorting. See the feature enablement tutorials for more details.
  • But you can also:
    • enable/disable sorting on individual columns
    • initialize the sorting directions on individual columns
    • control the sorting direction
    • use a sorting function
