Installation guides

Prerequisites

Server-side:

  • Java 6 or higher

Client-side:

  • jQuery v1.10.1 max (v2.x is not compatible with the DataTables version supported by Dandelion-Datatables)
  • DataTables v1.9.4

Guides

The installation guide defers, depending on the template engine you want to use in your web application:

JSP   or   Thymeleaf
