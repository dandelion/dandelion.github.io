Installation guides

Requirements

Any application that uses the Dandelion framework:

  • must use Java 6+
  • must be deployed inside a Servlet 2.5+ container

Known issues

  • The VFS protocol is not supported yet. As a consequence, you can't use Dandelion-Datatables with JBoss or WildFly. (#23)
  • You can't use the Dandelion-Datatables' dialect inside a Thymeleaf fragment (#28)
