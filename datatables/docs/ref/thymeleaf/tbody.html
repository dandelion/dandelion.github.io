<tbody> attributes

Usage

The <tbody> attributes allow you to configure the DataTables callbacks. 

<script>
   function myCustomCreateRowCallback(nRow, aData, iDataIndex) {
      // ...
   }
</script>

<table id="myTableId" dt:table="true">
   <thead>
      <tr>
         <th>Id</th>
         <th>Firstname</th>
         <th>Lastname</th>
         <th>City</th>
         <th>Mail</th>
      </tr>
   </thead>
   <tbody dt:cbk:createdrow="'myCustomCreateRowCallback'">
      <tr th:each="person : ${persons}">
         <td th:text="${person.id}">1</td>
         <td th:text="${person.firstName}">John</td>
         <td th:text="${person.lastName}">Doe</td>
         <td th:text="${person.address.town.name}">Nobody knows!</td>
         <td th:text="${person.mail}">john@doe.com</td>
      </tr>
  </tbody>
</table>
