<thead> attributes

The <thead> attributes are mainly used to configure plugins.

Usage

<table id="myTable" dt:table="true" dt:ext="fixedheader">
   <thead dt:fixedOffsetTop="40">
      <tr>
         <th>Id</th>
         <th>Firstname</th>
         <th>Lastname</th>
         <th>City</th>
         <th>Mail</th>
      </tr>
   </thead>
   ...
</table>
