<table> attributes

Usage

The <table> attributes allow you to configure the main features of DataTables, including DataTables enablement.

You can either use a "DOM" source, i.e. a Java collection previously set in the page or request scope or an "AJAX" source, i.e. using a web service that must return data formatted in JSON. 

<table id="myTableId" dt:table="true">
   <thead>
      <tr>
         <th>Id</th>
         <th>Firstname</th>
         <th>Lastname</th>
         <th>City</th>
         <th>Mail</th>
      </tr>
   </thead>
   <tbody>
      <tr th:each="person : ${myCollection}">
         <td th:text="${person.id}">1</td>
         <td th:text="${person.firstName}">John</td>
         <td th:text="${person.lastName}">Doe</td>
         <td th:text="${person.address.town.name}">Nobody knows!</td>
         <td th:text="${person.mail}">john@doe.com</td>
      </tr>
   </tbody>
</table>
