<td> attributes

Usage

All current <td> attributes allow you to configure the exported columns. 

<table dt:table="true" ...>
   ...
   <tbody>
      <tr>
         ...
         <td dt:csv="${bean.mail}">
             <a th:href="${'mailto:' + bean.mail}" th:text="${bean.mail}">john@doe.com</a>
         </td>
         ...
      </tr>
   </tbody>
</table>

Reference

Name Description Value(s) Default value Data source compatibility
dt:csv Processed Thymeleaf expression that will appear in the CSV export only DOM
dt:xml Processed Thymeleaf expression that will appear in the XML export only DOM
dt:pdf Processed Thymeleaf expression that will appear in the PDF export only DOM
dt:xls Processed Thymeleaf expression that will appear in the XLS export only DOM
dt:xlsx Processed Thymeleaf expression that will appear in the XLSX export only DOM
