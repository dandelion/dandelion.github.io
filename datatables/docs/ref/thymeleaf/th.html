<th> attributes

The <th> attributes allow you to configure a column.

Usage

<table dt:table="true" ...>
   <thead ...>
      <tr>
         <th>...</th>
         <th>...</th>
         <th dt:filterable="true">...</th>
         <th>...</th>
         <th>...</th>
      <tr>
   </thead>
   ...
</table>

Reference

Name Description Value(s) Default value Data source compatibility
dt:name Name to be given to the column. Mainly used when using AJAX source and server-side processing DOM AJAX
dt:property When using a DOM source, this is the name of the object's attribute of the collection being iterated on. When using an AJAX source, this is the name of the JSON property to be read from the data obtained by the AJAX call. DOM AJAX
dt:sortable Enable or disable sorting on column true | false true DOM AJAX
dt:sortInitDirection If the column is sortable, sets the default sorting direction asc | desc asc DOM AJAX
dt:sortInitOrder If the column is sortable, sets the order in which the sort should be initialized Any column index (starting from 0) DOM AJAX
dt:sortDirection Comma-separated list of directions to be used to control the sorting sequence Comma-separated list of 'asc' or 'desc' strings DOM AJAX
dt:sortType If the column is sortable, configures the type of sort to apply to the column natural | date | anti_the | alt_string | currency | filesize | formatted_number Type-based, internally guessed by DataTables DOM AJAX
dt:filterable Enable or disable filtering in the column. By default, an input field will be created in the tfoot section of the table true | false false DOM AJAX
dt:filterType If the column is filterable, confires the filter type select | input | number | number_range | date_range input DOM AJAX
dt:filterValues Name of a Javascript variable containing data used to populate the filtering select Either an array of values or an array of objects DOM AJAX
dt:filterMinLength Number of characters to be entered in the filtering input field before filtering is triggered Any number 0 (filtering is triggered on keyup) DOM AJAX
dt:filterDateFormat Date format to be applied to the datepicker when the filterType is set to "date_range" Any combination of 'dd', 'd', 'mm', 'm', 'yyyy', 'yy' with '-', '/' or '.' as separator mm/dd/yyyy DOM AJAX
dt:filterCssClass If the column is filterable, CSS class(s) to be applied on the filtering element (whose type depends on the filter type) DOM AJAX
dt:default Default string to be used if the value return from the property attribute is empty or null Empty string DOM AJAX
dt:searchable Enable or disable searching in the column. If false, the column won't be filtered by the global search field true | false true DOM AJAX
dt:renderFunction Name of a Javascript function that will be called to render the column when using an AJAX source Any String representing the name of an accessible Javascript function. Bundle special syntax allowed AJAX
dt:format MessageFormat pattern that will be used to format the value passed in the property attribute Any valid pattern DOM
dt:visible Enable or disable the display of the column. Note that even if the column is not visible, it is searchable true | false true DOM AJAX
dt:selector jQuery selector targeting any element that will be populated with a filtering element (depending on the filterType attribute). This attribute only makes sense when the filterPlaceholder attribute is set to none (i.e. for external filtering form) Any jQuery selector Empty string DOM AJAX
