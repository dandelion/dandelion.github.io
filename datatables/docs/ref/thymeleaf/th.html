<th> attributes

Usage

The <th> attributes allow you to configure each column. 

<table dt:table="true" ...>
   <thead ...>
      <tr>
         <th>...</th>
         <th>...</th>
         <th dt:filterable="true">...</th>
         <th>...</th>
         <th>...</th>
      <tr>
   </thead>
   ...
</table>
