ExtraJs tag

Usage

Specify the location of an extra file containing Javascript code which will be merged with the main Javascript generated file.

<datatables:table>
   ...
   <datatables:extraFile src="..." />
   ...
</datatables:table>

See the using extra Javascript files for more details.

Reference

Name Description Value(s) Default value Data source compatibility
src (required) Location of the file containing Javascript code DOM AJAX
insert Specify where the extra file content must be inserted in the main Javascript generated file

BEFOREALL: at the beginning of the main Javascript generated file

BEFORESTARTDOCUMENTREADY: just before the start of the jQuery .ready() function

AFTERSTARTDOCUMENTREADY: just after the start of the jQuery .ready() function

BEFORENDDOCUMENTREADY: just before the end of the jQuery .ready() function

AFTERALL: at the end of the main Javascript file
