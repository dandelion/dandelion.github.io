ExtraJs tag

Specify the location of an extra file containing Javascript code which will be merged with the main Javascript generated file.

Usage

<datatables:table>
   ...
   <datatables:extraJs src="..." />
   ...
</datatables:table>

See the extra Javascript files section for more details.

Reference

Name Description Value(s) Default value Data source compatibility
bundles (required) Comma-separated list of bundle to include in the current request DOM AJAX
