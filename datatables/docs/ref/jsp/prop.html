Prop tag

Usage

Locally override a global [Dandelion-DataTables configuration property](./ref.properties.html).

    
<datatables:table>
   ...
   <datatables:prop name="property.name" value="property.value" />
   ...
</datatables:table>

Reference

Name Description Value(s) Default value Data source compatibility
name (required) Name of the property to override DOM AJAX
value (required) Value of the property to override DOM AJAX
