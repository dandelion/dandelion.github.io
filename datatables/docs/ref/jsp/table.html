Table tag

Usage

The table tag allows you to display a HTML table thanks to a set of attributes and nested {% refjsp dt column %} tags.

You can either use a "DOM" source, i.e. a Java collection previously set in the page or request scope or an "AJAX" source, i.e. using a web service that must return data formatted in JSON. 

<datatables:table id="myTableId" data="${myCollection}" ...>
   <datatables:column title="Id" property="id" />
   <datatables:column title="FirstName" property="firstName" />
   <datatables:column title="LastName" property="lastName" />
   <datatables:column title="City" property="address.town.name" />
   <datatables:column title="Mail" property="mail" />
</datatables:table>
You will find in the bellowing table all the available attributes for this tag.

Note that:

  • you can also use dynamic attributes, i.e. attributes that are not officially declared in the TLD. This is convenient especially for all native HTML attributes. 
    <datatables:table cellspacing="5" ...>
   ...
</datatables:table>
  • you can use expression language (EL) in all attributes. 
    <datatables:table info="${...}" ...>
   ...
</datatables:table>
