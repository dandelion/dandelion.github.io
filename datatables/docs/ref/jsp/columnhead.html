ColumnHead tag

The <datatables:column> tag allows you to fill in a column header with a more complex content than a simple title.

Usage

It must be used inside the {% refjsp dt column %} tag.

For example: 

<datatables:table id="myTableId" data="${myCollection}" ...>
   <datatables:column>
      <datatables:columnHead>
         <!-- Some HTML that will appear in the column header... -->
      </datatables:columnHead>
      <!-- Some HTML that will appear in each column cell -->
   </datatables:column>
</datatables:table>

Reference

This tag has no attribute.
