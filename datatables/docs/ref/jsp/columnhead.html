ColumnHead tag

Usage

The ColumnHead tag allows you to fill in a column header with a more complex content than a simple title.

Here are the steps you need to follow to use this tag:

  1. Write anything inside the ColumnHead tag but remember to set the {% refjsp dt columnhead uid %} attribute, which is an arbitrary unique id for the column. 
    <datatables:columnHead uid="anyStringHere">
   <%-- Anything you want here! (e.g. a master checkbox) --%>
</datatables:columnHead>
  2. In the associated column, don't forget to set the uid again, with the same value you previously set in the uid attribute of the ColumnHead tag. 
    <datatables:column uid="anyStringHere">
   <%-- Anything you want here! (e.g. a checkbox) --%>
</datatables:column>
