Callback tag

Usage

Allows to execute a DataTables callback.

<datatables:table>
 ...
 <datatables:callback type="..." function="..." />
 ...
</datatables:table>

See the using callback for more details.

Reference

Name Description Value(s) Default value Data source compatibility
type (required) Type of callback. See the corresponding tutorial for more details. cookie | createdrow | draw | footer | format | header | info | init | predraw | row DOM AJAX
function (required) Name of an external Javascript function to execute as a callback DOM AJAX
