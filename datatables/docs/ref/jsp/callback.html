Callback tag

The <datatables:callback> tag allows you to configure one or multiple DataTables callbacks in the table.

Usage

<datatables:table>
 ...
   <datatables:callback type="..." function="..." />
 ...
</datatables:table>

See the callback section for more details.

Reference

Name Description Value(s) Default value Data source compatibility
type (required) Type of the callback cookie | createdrow | draw | footer | format | header | info | init | predraw | row DOM AJAX
function (required) Name of a Javascript function that will be called inside the configured callback. Any String representing the name of an accessible Javascript function. Bundle special syntax allowed. DOM AJAX
