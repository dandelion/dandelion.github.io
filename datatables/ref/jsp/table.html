Table tag

Usage

The table tag allows you to create a HTML table and describe its behaviour through a set of attributes that are listed below.

<datatables:table data="${myCollection}" ...>
   ...
</datatables:table>
You will find in the belowing table all the available attributes for this tag.

Note that:

  • you can also use dynamic attributes. This is convenient especially for all native HTML attributes. 
    <datatables:table cellspacing="5" ...>
   ...
</datatables:table>
  • you can use runtime expressions in all attributes. 
    <datatables:table info="${...}" ...>
   ...
</datatables:table>
