Scrolling

You may need to allow horizontal or vertical scrolling, e.g. when you have a wide table, with a large number of rows and/or columns to display. Let's see in this tutorial how to proceed.

Vertical scrolling

Using JSP

You can use the {% refjsp dt table scrollY %} table attribute. 

<datatables:table id="myTableId" data="${persons}" scrollY="200px" paginate="false">
   <datatables:column title="Id" property="id" />
   <datatables:column title="FirstName" property="firstName" />
   <datatables:column title="LastName" property="lastName" />
   <datatables:column title="City" property="address.town.name" />
   <datatables:column title="Mail" property="mail" />
</datatables:table>

Note that disabling table pagination is not mandatory, it will work just fine with pagination enabled as well!

Using JSP

You can use the {% reftl dt table dt:scrolly %} table attribute. 

<table id="myTable" dt:table="true" dt:scrolly="'200px'" dt:paginate="false">
   <thead>
      <tr>
         <th>Id</th>
         <th>Firstname</th>
         <th>Lastname</th>
         <th>City</th>
         <th>Mail</th>
      </tr>
   </thead>
   <tbody>
      <tr th:each="person : ${persons}">
         <td th:text="${person.id}">1</td>
         <td th:text="${person.firstName}">John</td>
         <td th:text="${person.lastName}">Doe</td>
         <td th:text="${person.address != null and person.address.town != null} ? ${person.address.town.name}">Nobody knows!</td>
         <td th:text="${person.mail}">john@doe.com</td>
      </tr>
   </tbody>
</table>

Horizontal scrolling

Using JSP

You can use the {% refjsp dt table scrollX %} table attribute. 

<datatables:table id="myTableId" data="${persons}" scrollX="120%">
   <datatables:column title="Id" property="id" />
   <datatables:column title="FirstName" property="firstName" />
   <datatables:column title="LastName" property="lastName" />
   <datatables:column title="City" property="address.town.name" filterable="true" />
   <datatables:column title="Mail" property="mail" filterable="true" />
</datatables:table>

<table id="myTable" dt:table="true" dt:scrollx="'120%'">
   <thead>
      <tr>
         <th>Id</th>
         <th>Firstname</th>
         <th>Lastname</th>
         <th>City</th>
         <th>Mail</th>
      </tr>
   </thead>
   <tbody>
      <tr th:each="person : ${persons}">
         <td th:text="${person.id}">1</td>
         <td th:text="${person.firstName}">John</td>
         <td th:text="${person.lastName}">Doe</td>
         <td th:text="${person.address != null and person.address.town != null} ? ${person.address.town.name}">Nobody knows!</td>
         <td th:text="${person.mail}">john@doe.com</td>
      </tr>
   </tbody>
</table>
