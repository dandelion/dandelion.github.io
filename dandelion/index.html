Overview

Dandelion Core is the fundation of the Dandelion framework. This is a runtime solution designed to make web development simpler in JVM-based environments.

Dandelion is mainly based on asset bundles (TODO) and on a servlet filter which automatically inject assets into your HTML pages. See the diagram below.
