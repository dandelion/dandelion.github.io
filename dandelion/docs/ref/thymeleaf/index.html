Thymeleaf dialect reference

The attributes of the Dandelion dialect allow you to interact with the bundle graph for a given request.

All the attributes can be used anywhere in your template, on any HTML tag.

Usage

<div ddl:bundle-includes="bundle1,bundle2">
   ...
</div>

Reference

Name Description Default value Corresponding configuration name
ddl:bundle-includes Comma-separated list of bundles to include in the current request bundle.includes
ddl:bundle-excludes Comma-separated list of bundles to exclude from the current request bundle.excludes
ddl:js-excludes Comma-separated list of JavaScript assets to exclude from the current request asset.js.excludes
ddl:css-excludes Comma-separated list of CSS assets to exclude from the current request asset.css.excludes
