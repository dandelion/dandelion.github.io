Bundle tag

Usage

The <dandelion:bundle> tag allows you to add or exclude bundles from the current request. 

<dandelion:bundle includes="bundle1" />

Note that:

  • you can use expression language (EL) in all attributes. 
    <dandelion:bundle includes="${...}" />

Reference

Name Description Default value Corresponding configuration name
includes Comma-separated list of bundles to include in the current request bundle.includes
excludes Comma-separated list of bundles to exclude from the current request bundle.excludes
