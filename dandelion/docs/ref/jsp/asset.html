Asset tag

The <dandelion:asset> tag allows you to exclude individual JS/CSS from the current request.

Usage

<dandelion:asset jsExcludes="asset1" />

Note that:

  • you can use expression language (EL) in all attributes. 
    <dandelion:asset jsExcludes="${...}" />

Reference

Name Description Default value Corresponding configuration name
jsExcludes Comma-separated list of JS to exclude from the current request asset.js.excludes
cssExcludes Comma-separated list of CSS to exclude from the current request asset.css.excludes
