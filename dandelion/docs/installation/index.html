Installation guide

Requirements

Any application that uses the Dandelion framework:

  • must use Java 6 (or later)
  • must be deployed inside any Servlet 2.5 (or later) container

Required steps

Irrespective of which template engine your web application is based on, there are some required steps to follow.

Step 1/3: Install the server-side dependencies

{% assign artifact = 'dandelion-core' %} {% include build-tools.html %}

Step 2/3: Configure the Dandelion filter

Add the Dandelion filter definition and mapping to the web.xml file: 

<filter>
   <filter-name>dandelion</filter-name>
   <filter-class>com.github.dandelion.core.web.DandelionFilter</filter-class>
</filter>
<filter-mapping>
   <filter-name>dandelion</filter-name>
   <url-pattern>/*</url-pattern>
</filter-mapping>

Step 3/3: Configure the Dandelion servlet

Add the Dandelion servlet definition and mapping to the web.xml file: 

<servlet>
   <servlet-name>dandelionServlet</servlet-name>
   <servlet-class>com.github.dandelion.core.web.DandelionServlet</servlet-class>
</servlet>
<servlet-mapping>
   <servlet-name>dandelionServlet</servlet-name>
   <url-pattern>/dandelion-assets/*</url-pattern>
</servlet-mapping>

You're done with the basic installation! With all the steps above, you can already enjoy using Dandelion. But you might be a bit limited.
