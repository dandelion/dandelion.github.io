Dandelion 0.1.0 change log04-22-2013

Core

  • Assets Stack Features: Assets stack management system
  • Configuration Accessor: Centralized configuration to access on all properties in `dandelion/dandelion*.properties`

JSP implementation

  • New {% refjsp ddl assets renderer %}, Rendering assets
  • New {% refjsp ddl assets scopes %}, Adding Scopes
  • New {% refjsp ddl assets excludedScopes %}, Excluding Scopes
  • New {% refjsp ddl assets excludedAssets %}, Excluding Assets

Thymeleaf implementation

  • New {% reftl ddl assets ddl:assets-stack %}, Rendering assets
  • New {% reftl ddl assets ddl:assets-scopes %}, Adding Scopes
  • New {% reftl ddl assets ddl:assets-excludedScopes %}, Excluding Scopes
  • New {% reftl ddl assets ddl:assets-excludedAssets %}, Excluding Assets
