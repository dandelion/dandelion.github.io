Toggling the development mode

Even if the development mode is enabled by default, note that you can toggle this mode in multiple ways.

As most of the configuration properties, you have several options to do so:

  • Using a configuration property: just add the following key/value pair to your dandelion.properties (TODO lien) file: 
    dandelion.mode=development
  • Using a filter initialization parameter in your web.xml file: 
    <filter>
   <filter-name>dandelion</filter-name>
   <filter-class>com.github.dandelion.core.web.DandelionFilter</filter-class>
   <init-param>
      <param-name>dandelion.mode</param-name>
      <param-value>development</param-value>
   </init-param>
</filter>
  • Using a system property: add the following system property to your server starting script: 
    -Ddandelion.mode=development

Remember that as for any other configuration property (TODO lien), the following rule applies: system property > filter init param > properties file.
