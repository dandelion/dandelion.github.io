MemoryAssetCache

The {% javadoc ddl dandelion-core MemoryAssetCache %} class is the default implementation used for caching.

Cache name default
Service provider {% javadoc ddl dandelion-core MemoryAssetCache %}

This implementation contains one map for asset caching and another for request caching. The max element of each of them can be configured thanks to the cache.asset.max.size and cache.request.max.size configuration properties.
