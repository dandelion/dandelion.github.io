Production mode

Dandelion provides two modes: development and production. We'll see here what it is hiding behind the development mode.

Introducing the developement mode

Dandelion has been designed to make the web development simpler. This is why the framework ships a development mode which helps developers making great apps faster thanks to the following features:

  • Asset reloading thanks to specific HTTP cache headers. Read more here. (TODO lien)
  • A bundle graph viewer, allowing you to visualize on runtime how Dandelion will manage your assets. Read more here. (TODO lien)

  • However, much work remains to be done. For example the bundle full reloading is not supported yet, which means each time you modify a bundle (TODO lien), you have to restart your server.
